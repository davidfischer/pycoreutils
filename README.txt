***********
PyCoreutils
***********

A pure-Python implementation of the GNU Coreutils.


Goals
-----

1. Have fun!
2. Have a Linux-like shell environment in Windows.
3. One day create a pure-Python Linux distribution.


Requirements
------------

PyCoreutils requires Python 2.6 or greater.


Development
-----------

You can get the latest version at https://launchpad.net/pycoreutils.


Haven't I seen this before?
---------------------------

David Cantrell also has a project with the same name at
https://github.com/dcantrell/pycoreutils. His project is a little different in
that it uses C-bindings, is GPL-licensed and is structured differently.


Warning
-------

This is alpha software, Not all the flags in the help-section are
implemented, and some behave different than one might expect.
